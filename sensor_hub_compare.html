應用選型建議
Application Fit · Which Sensor Hub to Choose?
100
SH-100 — 單點感測 / 基本應用
Single-channel · Basic Sensing

單通道 I²C 感測，適合單一種類感測器應用：溫度監測、濕度偵測、簡易開關量採集。成本最優，適合大量部署的簡單感測場景。

200
SH-200 — 多通道 / 進階感測
8-channel · Advanced Sensing

8通道 I²C/SPI 感測、12-bit ADC、可程式增益、Data Logging。適合冷鏈多點監測、振動分析、智慧建築 HVAC、工業設備預測性維護等複雜感測場景。
